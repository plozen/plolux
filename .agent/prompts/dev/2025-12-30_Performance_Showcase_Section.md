### 2. 텍스트 콘텐츠 (데이터)

- **하단 신뢰 뱃지**: "Google & Naver Search Optimized" (체크 아이콘 포함)

### 3. 핵심 비주얼 (Lighthouse Scoreboard)

4개의 원형 게이지(Gauge)가 동시에 차오르는 애니메이션을 구현하세요.

- **지표 항목 (Label)**:
  1. 성능 (Performance)
  2. 접근성 (Accessibility)
  3. 권장사항 (Best Practices)
  4. 검색엔진 최적화 (SEO)
- **애니메이션 로직**:
  - **Trigger**: 사용자가 스크롤하여 해당 섹션이 뷰포트에 들어왔을 때 실행 (`once: true`).
  - **Action**: 숫자가 `0`에서 `100`까지 빠르게 카운팅 (약 1.5초~2초 소요).
  - **Visual**: 원형 프로그레스 바(SVG Circle)가 숫자와 동시에 100%까지 차오름.
  - **Color**: PLOZEN 브랜드 컬러인 **Bio Lime (`#D6FF00`)**을 사용하여 형광 느낌 강조.

## 코드 품질 요구사항

- 클라이언트 컴포넌트(`"use client"`)로 작성하되, 최적화를 고려하세요.
- 코드는 깔끔하고 모듈화되어야 하며, 즉시 복사해서 사용할 수 있는 완성된 형태여야 합니다.
- **스타일링**: **SCSS Modules**와 Mixin(`@include mobile`)을 사용하여 구현하세요. **Tailwind CSS 클래스(`w-full`, `hidden` 등)는 절대 사용하지 마세요.**
- **테마 지원**: 다크 모드와 라이트 모드를 모두 완벽하게 지원해야 합니다. (`var(--bg-primary)`, `var(--text-primary)` 등 CSS 변수 활용)
- 모바일(화면 폭이 좁을 때)에서도 원형 차트가 찌그러지지 않도록 레이아웃을 신경 쓰세요.
