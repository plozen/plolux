# 1. 현재 상황 및 목표 (Context & Goal)

우리는 [Step 2]를 통해 App Shell 레이아웃, 하단 네비게이션, 그리고 기본 버튼 컴포넌트 구현을 완료했습니다.

이제 앱의 얼굴인 **홈 화면(Home Page)**을 완성해야 합니다.

당신의 목표는 사용자에게 압도적인 시각적 경험을 제공하는 **'실시간 랭킹 대시보드'**를 구축하는 것입니다. 실제 백엔드 연동 전이므로, **현실적인 Mock Data**를 사용하여 UI를 구성하십시오.

---

# 2. 실행 가이드: 단계별 작업 지시 (Step-by-Step Instructions)

### [Task 0] 필수 환경 구축: 다국어 & 다크모드 (Prerequisites)

UI 구현 전, 글로벌 서비스를 위한 핵심 인프라를 먼저 갖춥니다.

1.  **국제화 (i18n)**:
    - 설치: `pnpm add next-intl`
    - 구조 변경: `src/app/[locale]/` 라우팅 적용.
    - 언어셋: 한국어(`ko`), 영어(`en`) 지원. 단순 번역 JSON 파일(`messages/ko.json`) 생성.
2.  **다크/라이트 모드 (Theme)**:
    - 설치: `pnpm add next-themes`
    - 설정: `src/components/providers/ThemeProvider.tsx` 생성 (`attribute="class"` 사용).
    - 스타일: `_variables.scss`에서 `:root` (라이트)와 `[data-theme='dark']` (다크) 변수 분리 정의.
    - **기본값**: 시스템 설정 따르되, K-Pop 무드상 다크 모드가 우선.

### [Task 1] Mock Data 구조화 (Data Layer)

실제 데이터 구조를 정의합니다. **다국어 지원**을 고려해야 합니다.

파일 생성: `src/lib/mock-data.ts`
데이터 구조 (IdolType):

- `id`: string (uuid)
- `name`: `{ en: string, ko: string }`
- `company`: `{ en: string, ko: string }`
- `rank`: number
- `firepower`: number
- `change`: 'up' | 'down' | 'same'
- `image`: string (SVG Placeholder. **실사 금지**)

### [Task 2] 랭킹 아이템 컴포넌트 (Ranking Item)

리스트의 각 행을 담당하는 컴포넌트입니다.

파일 위치: src/components/features/ranking/RankingItem/ (index.tsx, RankingItem.module.scss)

기능 및 스타일 명세:

- **Layout**: 좌측에 순위와 변동 화살표(`lucide-react`), 중앙에 프로필(Gradient Circle)과 이름, 우측에 화력(Firepower)과 Support 버튼.
- **Top 3 Highlight**:
  - 1위: 금빛(Gold) Glow 효과 및 테두리.
  - 2위: 은빛(Silver).
  - 3위: 동빛(Bronze).
  - 4위 이하: Glassmorphism 기본 스타일.
- **Animation**: `framer-motion`을 사용하여 스크롤 시 화면에 부드럽게 진입(Fade In + Slide Up)해야 합니다.

### [Task 3] 홈 페이지 구성 (Page Composition)

`src/app/page.tsx`를 전면 수정하여 대시보드를 완성하십시오.

**구조**:

1. **Header**: `Kpop Company League` 로고 + **[Lang/Theme Toggle]**. (헤더 우측에 언어 변경(KR/EN) 및 다크모드 토글 버튼 배치).
2. **Hero Section**: "Current Top Firepower"를 강조하는 배너. (추상적인 3D 오브젝트나 팬아트 스타일 그래픽 사용).
3. **Ranking List**: [Task 1]의 Mock Data를 `map`으로 순회하여 [Task 2]의 `RankingItem`을 렌더링.
4. **Interaction**: 리스트 컨테이너에 `staggerChildren` 애니메이션을 적용하여 아이템이 순차적으로 뜨게 만드십시오.

# 3. 디자인 디테일 (Design Details)

K-Pop 특유의 화려함과 세련미를 살려야 합니다.

- **Rank Badge**: 순위 숫자는 `Montserrat` 폰트(Bold, Italic)를 사용하여 역동적으로 표현하십시오.
- **Profile Image**: 원형(Circle) 마스크 처리, 1위는 이미지 주변에 회전하는 그라데이션 보더(CSS Animation)를 추가하십시오.
- **Spacing**: 모바일 환경을 고려하여 터치 영역을 넉넉하게 잡고(최소 44px), 리스트 아이템 간 간격은 `12px`로 설정하십시오.
